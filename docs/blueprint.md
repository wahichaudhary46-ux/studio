# **App Name**: SparkMemo

## Core Features:

- User Authentication: Secure user registration and login using Google authentication as demonstrated in the provided code snippet.
- Note Creation & Editing: Allow authenticated users to create new notes with titles and content, and to edit their existing notes.
- Note Listing: Display a personalized list of all notes belonging to the logged-in user, accessible from a dashboard.
- Note Persistence: Store and retrieve all user notes securely in a Firestore database, as configured in the provided Firebase setup.
- AI-Powered Summarization Tool: Integrate a generative AI tool to automatically provide a concise summary of a note's content upon creation or on-demand, helping users quickly grasp key information.
- Note Deletion: Enable users to permanently delete notes they no longer need from their collection and the database.

## Style Guidelines:

- A clean and organized aesthetic with a light color scheme. The primary color is a deep, professional blue-grey (#4072BF) to convey clarity and technological sophistication. This hue helps anchor the design with a calm, authoritative presence, avoiding typical corporate drabness.
- The background color is a very light desaturated blue (#ECEFF3), derived from the primary hue, to provide a spacious and uncluttered canvas that enhances readability without being stark white. It ensures text and interactive elements stand out clearly.
- An accent color of vibrant cyan-teal (#1F89A8) will be used sparingly for interactive elements, highlights, and calls to action. Its contrasting brightness and saturation against the primary and background colors draw the user's attention effectively to important actions.
- For headlines and prominent UI elements, use 'Space Grotesk' (sans-serif) to impart a modern, technical, and slightly structured feel that aligns with an organized application. For body text, 'Inter' (sans-serif) will provide excellent readability and a neutral, objective aesthetic for note content.
- Employ a set of clear, outline-style icons. These will be simple and universally recognizable to enhance usability without adding visual clutter, maintaining the app's focus on organization and content.
- A minimalist, responsive layout emphasizing a clear content hierarchy. Notes will be presented in an easy-to-scan list or grid view, with ample whitespace. The design prioritizes content first, adapting seamlessly across desktop and mobile screens to ensure a consistent user experience.
- Subtle, functional animations will be used for state changes, such as saving a note or displaying a summary. These micro-interactions should be smooth and unobtrusive, providing clear feedback to the user without distracting from the core tasks.
